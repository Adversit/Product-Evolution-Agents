# EvoPM Agent · 演示视频 TTS 逐字稿

> 用法：把下面【朗读】里的文字逐段（或文末「连续朗读稿」整段）粘进**剪映 →「文本朗读」**生成 AI 配音；
> 录屏只录画面、不用出声，最后把配音轨拖到对应画面下对齐即可。
> 文字已避开 TTS 会读错的符号（箭头、斜杠、∥、× 等），数字与英文术语保持原样可正常朗读。

**剪映操作速记**：导入录屏 → 顶部「文本」→「新建文本」或直接「文本朗读」→ 粘贴本段文字 → 选音色（推荐「沉稳解说」或「知性女声」）→ 语速 1.0 → 生成 → 拖到画面下对齐。建议**按片段分 7 次生成**，方便和画面切换卡点对齐。

总时长目标 **4 分 30 秒**。录屏前先在 PowerShell 设 `$env:PYTHONWARNINGS="ignore"` 压掉 CLI 告警，全程用 replay 模式。
